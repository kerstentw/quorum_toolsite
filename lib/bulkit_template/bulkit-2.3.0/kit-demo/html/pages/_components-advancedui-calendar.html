Inline Calendar

Bulkit comes with a calendar extension that can be used in different flavours. The first one is an inline and small multipurpose calendar UI that you can use in your own pages and components. Edit the initial variables to change the calendar colors.

Calendar Picker New

Calendar Datepicker

The Calendar can behave like a datepicker when wired to a text input. It has to be initialized with Javascript.

Datepicker Modal

The Datepicker can be displayed as a modal, with an opaque overlay. The overlay option has to be set to true.
